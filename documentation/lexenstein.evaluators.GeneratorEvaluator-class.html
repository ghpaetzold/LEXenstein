evaluateGenerator(self, victor_corpus, substitutions)

source code 

Performs an intrinsic evaluation of a Substitution Generation approach.

Parameters:
  • victor_corpus - Path to a training corpus in VICTOR format. For more information about the file's format, refer to the LEXenstein Manual.
  • substitutions - A dictionary that assigns target complex words to sets of candidate substitutions. Example: substitutions['perched'] = {'sat', 'roosted'}
Returns:
Values for Potential, Precision, Recall and F-measure for the substitutions provided as input with respect to the gold-standard in the VICTOR corpus. For more information on how the metrics are calculated, please refer to the LEXenstein Manual.
