source code 

Creates an instance of the NorvigCorrector class.

Parameters:
  • model_file - Path to a file containing either raw, untokenized text, or a binary spelling correction model. If "model_file" is the path to a text file, then the value of "format" must be "text". If "model_file" is the path to a binary spelling correction model, then the value of "format" must be "bin".
  • format - Indicator of the type of input provided. Possible values: "text", "bin".

correct(self, word)

source code 

Returns the spell-corrected version of a word. If the model determines that the word has no spelling errors, it returns the word itself.

Parameters:
  • word - Word to be spell-corrected.

saveBinaryModel(self, model_path)

source code 

Saves the spelling correction model in binary format. The saved model can then be loaded with the "bin" format during the creation of a NorvigCorrector.

Parameters:
  • model_path - Path in which to save the model.
