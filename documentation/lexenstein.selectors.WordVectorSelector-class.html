selectCandidates(self, substitutions, victor_corpus, proportion=1.0, stop_words_file=None, window=99999, onlyInformative=False, keepTarget=False, onePerWord=False)

source code 

Selects which candidates can replace the target complex words in each instance of a VICTOR corpus.

Parameters:
  • substitutions - A dictionary linking complex words to a set of candidate substitutions Example: substitutions['perched'] = {'sat', 'roosted'}
  • victor_corpus - Path to a corpus in the VICTOR format. For more information about the file's format, refer to the LEXenstein Manual.
  • proportion - Percentage of substitutions to keep.
  • stop_words_file - Path to the file containing stop words of the desired language. The file must contain one stop word per line.
  • window - Number of tokens around the target complex sentence to consider as its context.
  • onlyInformative - If True, only content words are considered as part of the complex word's context, such as nouns, verbs, adjectives and adverbs.
  • keepTarget - If True, the complex target word is also included as part of its context.
  • onePerWord - If True, a word in the complex word's context can only contribute once to its resulting word vector.
Returns:
Returns a vector of size N, containing a set of selected substitutions for each instance in the VICTOR corpus.

Saves a set of selected substitutions in a file in VICTOR format.

Parameters:

  • substitutions - The vector of substitutions selected for the VICTOR corpus.
  • output_path - The path in which to save the resulting VICTOR corpus.
  • addTargetAsCandidate - If True, adds the target complex word of each instance as a candidate substitution.
