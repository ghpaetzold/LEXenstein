Returns:

getTrainingModel(self, features_file, c, epsilon, kernel, output_file)

source code 

Trains an SVM-Rank ranking model. The model produced can be used as the "model_file" parameter of the getScoresFile function.

Parameters:
  • features_file - Path to features file produced over a training VICTOR corpus. Should be produced by the getFeaturesFile function.
  • c - Trade-off between training error and margin. Recommended values: 0.001, 0.01
  • epsilon - Acceptable error margin. Recommended values: 0.00001, 0.0001
  • kernel - ID for the kernel to be used. Kernels available: 0 - Linear 1 - Polynomial 2 - Radial Basis Function 3 - Sigmoid
  • output_file - Path in which to save the resulting SVM-Rank model.

trainRankerWithCrossValidation(self, victor_corpus, folds, test_size, temp_folder, temp_id, Cs=['0.01', '0.001'], epsilons=[0.0001, 0.001], kernels=['0', '2', '3'])

source code 

Trains a SVM Ranker while maximizing hyper-parameters through cross-validation. It uses the TRank-at-1 as an optimization metric.

Parameters:
  • victor_corpus - Path to a training corpus in VICTOR format. For more information about the file's format, refer to the LEXenstein Manual.
  • folds - Number of folds to be used in cross-validation.
  • test_size - Percentage of the dataset to be used in testing. Recommended values: 0.2, 0.25, 0.33
  • temp_folder - Folder in which to save temporary files.
  • temp_id - ID to be used in the identification of temporary files.
  • Cs - Trade-offs between training error and margin. Recommended values: 0.001, 0.01
  • epsilons - Acceptable error margins. Recommended values: 0.00001, 0.0001
  • kernels - ID for the kernels to be considered. Kernels available: 0 - Linear 1 - Polynomial 2 - Radial Basis Function 3 - Sigmoid
