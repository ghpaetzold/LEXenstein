Creates a BiranGenerator instance.

Parameters:
  • mat - MorphAdornerToolkit object.
  • complex_vocab - Path to a vocabulary of complex words. For more information on how to create the file, refer to the LEXenstein Manual.
  • simple_vocab - Path to a vocabulary of simple words. For more information on how to create the file, refer to the LEXenstein Manual.
  • complex_lm - Path to a language model built over complex text. For more information on how to create the file, refer to the LEXenstein Manual.
  • simple_lm - Path to a language model built over simple text. For more information on how to create the file, refer to the LEXenstein Manual.
  • nc - NorvigCorrector object.
  • pos_model - Path to a POS tagging model for the Stanford POS Tagger. The models can be downloaded from the following link: http://nlp.stanford.edu/software/tagger.shtml
  • stanford_tagger - Path to the "stanford-postagger.jar" file. The tagger can be downloaded from the following link: http://nlp.stanford.edu/software/tagger.shtml

Generates substitutions for the target words of a corpus in VICTOR format.

Parameters:
  • victor_corpus - Path to a corpus in the VICTOR format. For more information about the file's format, refer to the LEXenstein Manual.
Returns:
A dictionary that assigns target complex words to sets of candidate substitutions. Example: substitutions['perched'] = {'sat', 'roosted'}
