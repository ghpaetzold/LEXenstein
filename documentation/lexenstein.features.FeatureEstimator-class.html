addCollocationalFeature(self, language_model, leftw, rightw, orientation)

source code 

Adds a set of collocational features to the estimator. The values will be the language model probabilities of all collocational features selected. Each feature is the probability of an n-gram with 0<=l<=leftw tokens to the left and 0<=r<=rightw tokens to the right. This method creates leftw*rightw+1 features.

Parameters:
  • language_model - Path to the language model from which to extract probabilities.
  • leftw - Maximum number of tokens to the left.
  • rightw - Maximum number of tokens to the right.
  • orientation - Whether the feature is a simplicity of complexity measure. Possible values: Complexity, Simplicity.

addLexiconFeature(self, lexicon, orientation)

source code 

Adds a lexicon feature to the estimator. The value will be 1 if a given candidate is in the provided lexicon, and 0 otherwise.

Parameters:
  • lexicon - Path to a file containing the words of the lexicon. The file must have one word per line.
  • orientation - Whether the feature is a simplicity of complexity measure. Possible values: Complexity, Simplicity.

addSentenceProbabilityFeature(self, language_model, orientation)

source code 

Adds a sentence probability feature to the estimator. The value will be the language model probability of each sentence in the VICTOR corpus with its target complex word replaced by a candidate.

Parameters:
  • language_model - Path to the language model from which to extract probabilities.
  • orientation - Whether the feature is a simplicity of complexity measure. Possible values: Complexity, Simplicity.
